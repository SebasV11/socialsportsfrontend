# Image Assets

Gebruik deze structuur voor app-afbeeldingen:

- `landing/` voor onboarding en startschermen
- `auth/` voor login/register visuals
- `shared/` voor herbruikbare iconen/illustraties

Aanbevolen bestandsnamen:

- `landing/football-player.png` (506x494)
- `landing/hero-bg-shape.png` (optioneel)

Tip: gebruik lowercase en koppeltekens in bestandsnamen.
